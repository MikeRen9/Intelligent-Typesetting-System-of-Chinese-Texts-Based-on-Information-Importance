# Intelligent-Typesetting-System-of-Chinese-Texts-Based-on-Information-Importance
## 简介

这是一个基于信息重要程度的智能化中文排版系统，你可以将需要排版的文本输入系统，将会得到新的基于信息重要程度的文本排版。

## 配置

* python
* flask

## 怎样使用

1. 下载仓库
2. 运行methods.py
3. 打开index.html
4. 输入文本，开始排版！
